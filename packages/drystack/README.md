# @drystack/core

Core engine for drystack — a git-backed content management system (fork of [Keystatic](https://github.com/Thinkmill/keystatic)). Provides the form/field system, content reader (local filesystem or GitHub), and admin UI building blocks used by framework integrations such as [`@drystack/astro`](https://www.npmjs.com/package/@drystack/astro).

## Install

```sh
npm install @drystack/core
```

This package is normally installed as a dependency of a framework integration package rather than used directly.

## Documentation

See the [drystack repository](https://github.com/khancoder282/drystack) for configuration and usage.
